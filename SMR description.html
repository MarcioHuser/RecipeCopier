Copy your recipes by pointing and clicking!



With this mod you can select a recipe from a machine and replicate it along multiple machines. You can also aim to a manufacturer and check wich recipe it currently has defined.

Multiplayer compatibility still pending

Find me at Satisfactory Modding Discord server (MarcioHuser#2377)

What it includes:

A hand tool with a screen that allow you:
  • To see the recipe defined at the machine you are aiming;
  • To select a recipe from a machine by clicking on it while holding Left Ctrl;
  • To define a selected recipe into new machines that can accept it, bi clicking;
